🔌 Koneksi TikTok Live
Cara dapat Session Cookie:
1. Login TikTok di Chrome/Firefox (bukan HP)
2. Buka tiktok.com → tekan F12
3. Tab Application → Cookies → tiktok.com
4. Cari nama sessionid → copy nilainya
5. Paste di kolom di atas (cukup nilai tanpa "sessionid=")

Kenapa perlu? TikTok membatasi akses tanpa login. Session cookie membuktikan akunmu sudah login.
